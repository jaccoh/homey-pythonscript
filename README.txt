Python Script lets you write and run Python code directly from Homey Advanced Flows. Write a quick inline script in a flow card, or build more complex automations using any pip package — requests, numpy, pandas, or anything else from PyPI. Scripts can read logic variables, control device capabilities, and trigger other flows from Python.

A built-in IDE on the app settings page lets you save named scripts with a Monaco editor, manage virtual environments, and run scripts directly from the browser with live output. Named scripts can be called from any flow card, keeping your flows clean while the logic lives in one place.
